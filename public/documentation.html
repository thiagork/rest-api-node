Documentation

This API has the following endpoints:

  • /movies GET Returns a list of All movies
  • /movies/:title GET Returns data about a single movie
  • /movies/genres/:genre GET Returns data about a genre
  • /directors/:director GET Returns data about a director
  • /users POST Allows new users to be registered. Expects a body with an object containing "name", "email", "dateOfBirth"(string), favoriteMovies(array of strings)
  • /users/:name/:email/:dateOfBirth PUT Allows users to update their info
  • /users/:name/favorites POST Allows user to add favorite movies. Expects a body with an object containing "movie" with 1 movie listed as string.
  • /users/:name/favorites DELETE Allows users to remove a movie from their list of favorites
  • /users/:name DELETE Allows a user to deregister
  • /documentation GET Serves this documentation file
